# Contributing to LearnTogether-Grow

Welcome! We are thrilled that you want to contribute to **LearnTogether-Grow**. Whether it's fixing a bug, improving documentation, or adding a new exercise, your help is appreciated.

## How to Get Started
1. **Find an Issue:** Browse the [Issues](https://github.com/TScom-dew/LearnTogether-Grow/issues) tab. Look for tags like `good first issue` if you are `new`.
2. **Fork the Repo:** Click the "Fork" button at the top right of the repository page.
3. **Clone your Fork:**

   
```bash
git clone https://github.com/[user-name]/[repo-name].git

git clone https://github.com/TScom-dew/LearnTogether-Grow.git
```
4. **Create a Branches:**
```Bash
git checkout -b [Your-branch-name]

git checkout -b feature
```

5. **Make changes:** Add your code or documentation updates.
6. Commit & Push:
```bash
git add .
git commit -m "Add descriptive message about your changes"
git push origin [Your-branch-name]
```
7. **Submit a Pull Request:** Open a Pull Request from your branch to the main branch of this repository.


---

## Contribution Guidelines
- **Be Kind:** This is a learning space. Please be respectful and supportive in code reviews.

- **Code Style:** Keep your code clean and add comments where necessary, especially for complex logic in C/C++.

- **Direct Contributions:** You are welcome to add new C/C++ programs directly without creating an issue first! **Please ensure you place your work in a new, uniquely named folder to avoid merge conflicts.**


## Need Help?
If you get stuck, feel free to comment on the issue you are working on or reach out!

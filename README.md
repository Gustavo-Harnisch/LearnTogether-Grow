# Learn-together-grow 🌱

**A community-driven repo for learning through contribution and code review.**

This repository is created specifically for those who wants to contribute, learn, and grow together. Feel free to contribute.

### 🛠️ Tech Stack
`C` `C++` `DSA` `HTML` `CSS` `JavaScript`

 **Feel free to ask questions related to tech stack**
 
### 🎯 Purpose
The aim of this repo is to provide a safe space where anyone can contribute code and learn in the process  togather. 

### 🚀 How It Works
1. **Pick an Issue** - Check the `Issues` tab for problems to solve or you can also add your program in given teck stack
2. **Fork & Code** - Write your solution in given tech stack
3. **Pull Request** - Submit your code 
4. **Get Feedback** - Receive detailed review as soon as possible
5. **Learn & Improve** - Update your code based on feedback

### 🤝 Who Is This For?
Beginners, students, and self-learners. No experience required. Mistakes are welcome - that's how we learn.

**Let's build skills together. One PR at a time.**
